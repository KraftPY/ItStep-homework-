• В чем отличие LocalStorage от SessionStorage?

    LocalStorage и SessionStorage - это объекты для хранения строковых данных на устройстве пользователя. Отличаются эти два объекта, тем что записаные данные через SessionStorage удаляются автоматически после закрытия браузера, окна или вкладки, но остаются при перезагрузки страницы, а когда данные записаны через LocalStorage, то эти данные остаются на все время и удаляются только вручную.

  • Где на компьютере хранятся Cookie?

    c:\users\%userprofile%\AppData\ - каждый браузер создает свою папку, где и хранит временные файлы

  • Зачем нужен шаблон SingleTone?

    Этот шаблон нужен, для создания проектов где используется один экземпляр класса. Чтоб при создании еще одного объекта этого класса, возвращалась не новая ссылка на новый экземпляр, а возвращалась ссылка уже созданого единственного экземпляра.

  • Почему шаблоны проектирования разделяются на три категории?

    Потому что все шаблоны проектирования выполняют три основные функции создание объектов, изменение объектов и взаимодействие между объектами. Поэтому их зазделили на три категории:
    1) Порождающие
    2) Структурные
    3) Поведенческие

  • Когда и кто придумал MVC?

    MVC была придумана Трюгве Реенскаугом (Trygve Reenskaug) когда он работал в Xerox PARC в 1978/79 годах.

    Model-View-ViewModel, Model-View-Presenter, Hierarchical model–view–controller

  • Как работают get/set операторы в описании класса?

    Операторы setter и getter связывают свойство объекта с методом. Чтоб метод стал свойством, перед названием метода пишется set или get. Метод обозначенный get возвращает какое-то подсчитаное значение, а set устанавливает полученное значение после проверки (если такая проверка есть).
