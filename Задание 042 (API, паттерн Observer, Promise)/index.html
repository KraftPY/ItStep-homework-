• Что такое API и зачем он нужен?

    • API (Application Programming Interface) - это уже написанный рабочий код (набор функций, классов) для взаимодействия с другой программой (приложением) или для получения нужного тебе результата с помощью специального API, при этом не реализовывая весь функционал этого API у себя в коде.

  • Зачем нужен pattern Obsever?

    • Позволяет следить за изменениями объектов и уведомлять другие объекты, которые на эти объекты «подписанны», если произошли какие-то изменения.

  • Приведите пример Obsever`a?

    Facebook, Twitter, Youtube и тд. Интерактивная таблица с данными по которой выводятся различные виды графиков.

  • Что такое Promise?

    Promise - это встроенный класс объекта, с помощью которого можно обработать результат выполнения асинхронной операции.

  • Что такое RegExp?

    RegExp - это встроенный класс объекта с двумя методами exec и test, для работы с регулярными выражениями.

  • Зачем нужен RegExp?

    • Для поиска или замены нужной подстроки в тексте, которая задается в шаблоне символами или метасимволами.
