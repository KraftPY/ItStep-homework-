1. В чем отличие document от window?

Window - это глобальный объект окна в браузере, который хранит глобальные переменные, функции, историю. Document - это свойство Window, через которое можно добраться до всех объектов страницы (DOM), для их изменения или добавления новых.

2. За что отвечает свойство outerHTML?

outerHTML - это свойство объектов DOM, в котором содержится HTML элемента целиком.

3. Какие методы есть у элемента Element.classList и какие действия можно с ними производить?

element.classList - это свойство объектов DOM, через которое можно получить массив классов объекта. У возвращаемого этим свойством псевдо-массив, есть методы:
- add("class")/remove("class") - добавляет/удаляет класс объекту;
- contain("class") - возвращает истину/ложь, если класс найден/не найден;
- toggle("class") - если такого класса нет, то добавляет, а если есть, то удаляет;
- item(number) - доступ к классу под номером number в массиве;
