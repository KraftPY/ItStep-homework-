1. Кто и когда придумал XML?

    2. XML разработан консорциумом World Wide Web в комитете, возглавляемом Джоном Босаком (Jon Bosak) в 1998 году. Он является упрощенной версией SGML.

  2. Зачем нужен XML?

    3. В настоящее время XML широко используется для передачи данных между программами, он структурирует данные, организовывая их в легкую для разбора форму.

  3. Почему, XML хуже(или лучше) чем JSON?

    4. Если сравнивать их как структуру для передачи данных, то JSON лучше чем XML. Работать с объектной структурой намного легче, чем с DOM.

  4. В чем схожесть XML vs HTML ?

    5. Они оба являются Web языками разметки.

  5. Могу ли я работать с XML через getElementById?

    6. Да, так как это Web язык разметки у которого есть своя DOM структура и если у xmlElement будет задан атрибут id="", то при помощи getElementById можно будет получить этот элемент.
