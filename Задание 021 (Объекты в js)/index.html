Задание № 1. Драконья стая.

Напишите приложение по генерации драконьей стаи. Пользователь вводит количество драконов в стае. Сгенерировать массив объектов со случайными драконами.

  • Имя дракона
  • Локация, откуда прилетел дракон
  • Пол дракона(М/Ж)
  • Цвет дракона(7 цветов)
  • Возраст дракона(100 до 3000)
  • Головы дракона(1-20)

Входные данные:

Число - целое число от 1 до 100. Количество драконов в стае.

Выходные данные:

Массив объектов(стая). Суммарная сумма голов дракона, мощь стаи драконов, количество драконов мужского пола и количество драконов женского пола. Информацию по самому молодоу дракону и информацию по самому старому дракону. Среднее количество голов у драконов.

Задание № 2. Крестики-нолики. Начало.

Создайте веб-приложение, в котором есть двумерный массив 3 на 3 поля, заполненный буквами "Х" и "О". Выведите его в консоль.

Выходные данные:

Строка - преобразованный в строку массив

Выходные данные:
