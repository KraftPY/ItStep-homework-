1. Что делает тег picture и как его использовать в тандеме с тегами source img?

Тег <picture> нужен для отображения адаптивных изображений. Внутри тега <picture> используют один или несколько тегов <source>, где указывают изображение которое будет отображаться при заданных условиях внутри этого тега и тег <img> в котором задают default изображение, если не одно изображение из тегов <source> неподойдет.

2. С какими аттрибутами и тегами использовать тег optgroup?

Тег <optgroup> используют с аттрибутами disable (блокирует доступ к группе) и label (наименование группы, которое будет отображаться в выпадающем списке). Тег <optgroup> используется внутри тега <select>, а внутри самого тега <optgroup> используют тег <option>.

3. Опишите особенности использования тега video?

Тег <video> предназначен для воспроизведения видео контента на веб странице. Ссылка на видео указывается в атрибуте src или внутри тега <video> в теге <source>. Так как разные браузеры поддерживают разные видео-кодеки, то тег <source> можно использовать несколько раз и указать в них видео во всех форматах.

4. Что можно сделать с помощью тега audio?

При помощи тега <audio> можно воспроизвести аудио контент на веб странице. Так как и в теге <video>, ссылка на аудио файл указывается в атрибуте src или внутри тега <audio> в теге <source>. Так как разные браузеры поддерживают разные аудио-кодеки, то тег <source> можно использовать несколько раз и указать в них аудио файлы во всех форматах.

5. Какие возможности открывает использование тега iframe/frame?

Тег <frame> предназначен для разбиения веб страницы на фреймы, а в каждом фрейме можно загрузить отдельную веб страницу. Тег <iframe> служит для вставки внешнего фрейма на страницу, типичным примером является вывод видео с youtube.
