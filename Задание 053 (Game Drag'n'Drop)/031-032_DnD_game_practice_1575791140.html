Домашнее задание №031-032 - DnD - игра

Практика

Описание

  • Каждые 2-4(случайно) секунды происходит генерация нового html-элемента(Печенька) на экране в случайном месте(кроме центра экрана)
  • Элемент Печенька при генерации случайно получает один из трех внешних видов:
    • Желтый круг(Солнце) - 30%
    • Серый полукруг(Луна) - 60%
    • Синий ромб(Звезда) - 10%
  • В центре экрана находится круглый объект размером 100*100(Черная Дыра)
  • С начала игры запускается таймер с отметки в 5 секунд
  • Каждый занесенный объект в Черную Дыру увеличивает время в таймере на 4 секунды
  • По окончанию таймера объекты перестают генерироваться и их нельзя больше переносить
  • Каждый объект при попадании в Черную Дыру удаляется и увеличивает количество очков:
    • Луна - 3 очка
    • Солнце - 7 очков
    • Звезда - 11 очков
  • В черной дыре отображается время таймера и количество очков полученных игроком
  • Любая Печенька отображается на экране только 7 секунд. Если её не перенесли в Черную Дыру, она удаляется

Советы:

Запаковывайте каждый новый элемент, его время жизни, метод для смерти, ссылку на печеньку в отдельный объект. Объекты заносите в массив.

Формат работы

  • html, js, css файлы
